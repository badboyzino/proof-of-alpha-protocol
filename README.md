# Proof of Alpha Protocol

Proof of Alpha is a decentralized reputation protocol designed to track and verify crypto research.

In the current Web3 ecosystem, many people claim to have "alpha", but there is no transparent system to verify early research or predictions.

Proof of Alpha records research calls and evaluates them over time.

## Core Features

- Alpha call registry
- Reputation scoring system
- Transparent research history
- On-chain verification

## Vision

Create the credibility layer for crypto research in Web3.
